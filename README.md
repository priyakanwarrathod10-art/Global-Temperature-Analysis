# Global-Temperature-Analysis
Excel-based analysis of Global Temperature Change dataset including data cleaning, trend analysis, and dashboard visualization.

#   Global Temperature Change Analysis

# Overview

This project analyzes the Global Temperature Dataset using Microsoft Excel to identify long-term climate trends and temperature variations. The dataset was cleaned, structured, and visualized through an interactive dashboard.

# Tools & Techniques

Microsoft Excel (.xlsm)

Data Cleaning & Preprocessing

Pivot Tables

Charts & Dashboard Visualization

# Key Insights

Year-wise temperature trend analysis

Identification of global warming patterns

Visual representation of climate change indicators
